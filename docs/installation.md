# Installation

## From PyPI

```bash
pip install image-generation-mcp
```

## From Docker

```bash
docker pull ghcr.io/pvliesdonk/image-generation-mcp:latest
```

## From source

```bash
git clone https://github.com/pvliesdonk/image-generation-mcp
cd image-generation-mcp
uv sync --all-extras --dev
```
